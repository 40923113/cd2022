建立倉儲

先尋找老師倉儲的cmsimde site 

接著複製老師的網址

點選右上角+號裡的import a repository

之後將老師網址貼上到紅框的地方

籃框的部分打cd2022(名稱)

接著回到小黑窗打上

cd tmp

git clone --recurse-submodules https://github.com/40923113/cd2022.git

cd cd2022
